# TP 1 Calcul des Proposition

## Generalités

### Sous formules

- (a ∨ b) -> a ∨ b
- (a ∧ b) -> a ∧ b
- (e => f) -> e => f
- (e <=> f) -> e <=> f
- ¬i -> ¬i
- ((a => b) => b) => ¬(a ∨ b) -> a => b, a ∨ b, (a => b) => b, ¬(a ∨ b), ((a => b) => b) => (¬(a ∨ b))

### Mise sous forme normale conhectives

(( a => (b ∨ c)) ∧ ((b ∨ d) => (e ∧ f))) ∧ ¬e

-> ( (¬a ∨ b ∨ c) ∧ ( ¬(b ∨ d) ∨ (e ∧ f) ) ) ∧ ¬e
-> ( ¬e ∧ (¬a ∨ b ∨ c) ) ∧ ( ¬e  ∧ ( (e ∨ (¬(b ∨ d))) ∨ (f ∧ ¬(b ∨ d)) ))
-> ( ¬e ∧ (¬a ∨ b ∨ c) ) ∧ ( ¬e ∧ (e ∨ (¬(b ∨ d)) ) ∨ ¬e  ∧ (f ∧ ¬(b ∨ d)))
-> ( ¬e ∧ (¬a ∨ b ∨ c) ) ∧ ( (¬e ∧ ¬(b ∨ d)) ∨ (¬e ∧ f ∧ ( ¬e ∧ ¬(b ∨ d))))
-> ( ¬e ∧ (¬a ∨ b ∨ c) ) ∧ ()
